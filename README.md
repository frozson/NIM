# NIM

A Image Processing Module that uses CNN and other possibly useful AI to implement various features.

![Image Loading using Raylib](./docs/First_look.jpg)

## How to Install

Warning : It is still in Eary Development , so please notice on any issues and who want to experiment on it are please be mindful of what you are doing.

Run the following commands in your terminal: (Windows terminal command is given below)

#### First clone the repo to your local directory
```shell
git clone --recursive https://github.com/frozson/NIM.git

```
#### Go to the cloned repository and run

```shell
mkdir build
cd build
cmake ..
```
Visual Studio build in cmake can run it inside and produce an executable

Note : Now project is build as executable for users to check how it works.
Future Release will be a library and they have seperate examples
